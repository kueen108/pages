1. 개인정보처리방침 개요

인생연금 720(이하 "앱")은 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 등 관련 법령을 준수하고 있습니다.

본 개인정보처리방침은 앱이 제공하는 연금복권720+ 번호 분석 및 추천 서비스와 관련하여 이용자의 개인정보를 어떻게 수집, 이용, 보관, 보호하는지에 대해 설명합니다.

2. 수집하는 개인정보

2.1 자동으로 수집되는 정보

  • 기기 정보: 기기 모델, 운영체제 버전, 고유 기기 식별자
  • 앱 사용 정보: 앱 버전, 사용 시간, 기능 사용 패턴
  • 분석 데이터: 선호하는 번호 패턴, 사용 빈도, 기능별 사용 통계

2.2 사용자가 제공하는 정보

  • 저장된 번호: 사용자가 저장한 복권 번호 (로컬 저장)
  • 스캔 기록: QR 코드로 스캔한 복권 정보 (로컬 저장)
  • 설정 정보: 알림 설정, 분석 기간 설정 등

2.3 수집하지 않는 정보

본 앱은 다음 정보를 수집하지 않습니다:

  • 이름, 연락처, 이메일 등 개인 식별 정보
  • 금융 정보 또는 결제 정보
  • 위치 정보
  • 카메라 촬영 이미지 (QR 스캔은 실시간 처리 후 즉시 삭제)

3. 개인정보 이용 목적

수집된 정보는 다음과 같은 목적으로만 사용됩니다:

🎯 서비스 제공

  • 번호 추천 알고리즘 개선
  • 당첨 확인 기능 제공
  • 통계 분석 서비스 제공

📊 서비스 개선

  • 사용자 경험 최적화
  • 버그 수정 및 성능 개선
  • 새로운 기능 개발

🔔 알림 서비스

  • 추첨 결과 알림
  • 저장 번호 당첨 알림
  • 앱 업데이트 안내

4. 개인정보 보유 및 이용 기간

정보 유형 보유 기간 삭제 방법
저장된 번호 사용자가 직접 삭제 시까지 앱 내 삭제 기능 또는 앱 삭제
스캔 기록 최대 1년 자동 삭제 또는 수동 삭제
사용 통계 최대 6개월 자동 삭제
기기 정보 앱 사용 기간 동안 앱 삭제 시 자동 삭제

5. 개인정보 제3자 제공

원칙: 인생연금 720은 이용자의 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우에는 예외적으로 제공될 수 있습니다:

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

5.1 제3자 서비스 이용

앱은 다음과 같은 제3자 서비스를 이용할 수 있습니다:

  • Google Analytics: 앱 사용 통계 분석 (익명화된 데이터만 전송)
  • Firebase Crashlytics: 앱 안정성 모니터링 (충돌 보고서)

6. 개인정보 보호 조치

인생연금 720은 이용자의 개인정보를 안전하게 보호하기 위해 다음과 같은 조치를 시행합니다:

🔐 기술적 보호조치

  • 데이터 암호화 저장
  • 보안 프로토콜 사용 (HTTPS)
  • 정기적인 보안 업데이트

🏢 관리적 보호조치

  • 최소한의 정보만 수집
  • 접근 권한 관리
  • 정기적인 보안 점검

💾 물리적 보호조치

  • 데이터는 사용자 기기에만 저장
  • 서버 전송 최소화
  • 로컬 데이터 격리 저장

7. 이용자의 권리

이용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

✅ 열람권

📝 정정권

잘못된 정보 수정 요청

🗑️ 삭제권

개인정보 삭제 요청

  2. 앱 삭제를 통한 모든 데이터 제거
  3. 고객센터를 통한 요청

8. 아동의 개인정보 보호

만 14세 미만 아동 이용 제한
본 앱은 만 14세 미만 아동의 이용을 제한하고 있습니다. 만 14세 미만 아동의 개인정보를 수집하지 않으며, 실수로 수집된 경우 즉시 삭제하겠습니다.

9. 개인정보처리방침 변경

개인정보처리방침이 변경되는 경우, 최소 7일 전에 앱 내 공지사항을 통해 변경 내용을 사전 고지합니다.

중요한 변경사항이 있는 경우, 푸시 알림 또는 팝업을 통해 별도로 안내드립니다.
