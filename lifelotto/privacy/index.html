인생로또 645

개인정보처리방침

← KokomaSoft 홈으로 ← 인생로또 645로

시행일자: 2024년 9월 1일

KokomaSoft (이하 "회사")는 「인생로또 645」 앱 서비스(이하 "서비스")를 제공함에 있어, 「개인정보 보호법」 등 관련 법령에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 하기 위하여 다음과 같이 개인정보처리방침을 수립·공개합니다.

1. 개인정보의 수집 및 이용

📌 중요 안내

인생로또 645는 사용자의 개인정보를 수집하지 않습니다.

모든 로또 번호 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

수집하는 정보

  • 익명 사용 데이터: Firebase Analytics를 통한 앱 사용 패턴 (개인 식별 불가)
  • 익명 광고 추적 데이터: AdMob을 통한 광고 성과 측정 (개인 식별 불가)

수집 방법

  • 앱 사용 중 자동으로 생성되는 익명화된 통계 데이터
  • 제3자 서비스(Firebase, AdMob)를 통한 자동 수집

2. 개인정보의 이용 목적

수집된 익명 데이터는 다음의 목적으로만 이용됩니다:

  • 앱 기능 개선 및 오류 수정
  • 사용자 경험 향상을li>
  • 앱 성능 최적화
  • 맞춤형 광고 제공 (익명 데이터 기반)
  • 서비스 이용 통계 분석

3. 개인정보의 보유 및 이용 기간

로컬 데이터

  • 로또 번호 데이터: 앱 삭제 시까지 기기에 보관
  • 앱 설정 정보: 앱 삭제 시까지 기기에 보관

익명 분석 데이터

  • Firebase Analytics: Google 정책에 따라 14개월 또는 26개월 보관
  • AdMob: Google 광고 정책에 따라 보관

💡 참고: 앱을 삭제하면 기기에 저장된 모든 로컬 데이터가 자동으로 삭제됩니다.

4. 제3자 서비스 이용

서비스 제공을 위해 다음의 제3자 서비스를 이용합니다:

Firebase (Google)

  • 목적: 앱 사용 통계 분석, 충돌 보고서 수집
  • 수집 항목: 익명화된 사용 패턴, 기기 정보 (개인 식별 불가)
  • 개인정보처리방침: Firebase 개인정보처리방침

AdMob (Google)

  • 목적: 광고 표시 및 수익화
  • 수집 항목: 광고 ID, 기기 정보 (개인 식별 불가)
  • 개인정보처리방침: Google 개인정보처리방침

5. 개인정보의 안전성 확보 조치

회사는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  • 기술적 조치: iOS 샌드박스 내 안전한 저장소 사용
  • 통신 보안: HTTPS 프로토콜 사용
  • 데이터 최소화: 개인 식별 가능 정보 미수집
  • 접근 제한: 앱 내부 데이터는 해당 앱에서만 접근 가능

6. 이용자의 권리와 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:

데이터 삭제

  • 앱 삭제를 통한 모든 로컬 데이터 즉시 삭제
  • 설정 > 일반 > iPhone 저장 공간에서 앱 데이터 삭제

광고 추적 제한

  • iOS 설정 > 개인정보 보호 > 광고 > 광고 추적 제한

분석 데이터 수집 거부

  • iOS 설정 > 개인정보 보호 > 분석 및 개선 > 앱 개발자와 공유 해제

7. 아동의 개인정보 보호

회사는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만약 만 14세 미만 아동의 개인정보가 수집된 것을 확인한 경우, 즉시 삭제 조치를 취하겠습니다.

8. 쿠키 및 추적 기술

인생로또 645 앱은 쿠키를 사용하지 않습니다. 단, 제3자 서비스(Firebase, AdMob)가 서비스 제공을 위해 필요한 추적 기술을 사용할 수 있습니다.

9. 개인정보의 국외 이전

제3자 서비스 이용으로 인한 데이터 처리:

이전받는 자 이전 국가 이전 목적 이전 항목
Google LLC 미국 앱 분석 및 광고 서비스 익명화된 사용 데이터

10. 개인정보처리방침의 변경

이 개인정보처리방침은 2024년 9월 1일부터 적용됩니다.

회사는 법령이나 서비스의 변경 사항을 반영하기 위한 목적 등으로 개인정보처리방침을 수정할 수 있습니다. 개인정보처리방침이 변경되는 경우 변경 사항을 앱 내 공지사항을 통해 공지하겠습니다.
